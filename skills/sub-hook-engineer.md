# Sub-skill: hook-engineer

## Purpose
Generate and score 5+ opening hooks against the 3-Second Hook Rule, curiosity gap theory, and pattern-interrupt principles. This stage transforms the audience profile into concrete hook options, evaluates each against proven frameworks, and delivers a ranked set of hook implementations ready for retention scoring.

## When the Harness Calls This
Stage 2 of the `shortform-video-script-builder` main workflow, immediately following audience analysis. Must complete before research/evidence gathering stage.

## Inputs
- **Primary:** User's video script or content concept
- **From Stage 1:** Audience profile (platform psychology, 4-dimensional analysis, hook-pacing matrix)
- **Framework Reference:** Hook-Retention-Payoff (HRP) model hook phase, 3-Second Hook Rule, Curiosity Gap theory, Pattern Interrupt theory

## Procedure

### 1. Hook Strategy Selection
Based on audience profile, select primary hook strategies from this framework:

| Hook Strategy | Best For | Platform Fit | Complexity | Evidence Base |
|----------------|----------|--------------|------------|---------------|
| **Visual Pattern Interrupt** | TikTok, Reels (visual-first platforms) | High | Low | Visual attention research |
| **Audio-Vocal Hook** | All platforms, educational content | Universal | Medium | Auditory processing studies |
| **Curiosity Gap** | YouTube Shorts, educational | Medium | High | Information gap theory |
| **Controversial Statement** | High-intent audiences, opinion content | Medium | High | Cognitive dissonance research |
| **Value Proposition** | Problem-solving intent, utility content | Universal | Low | Value expectancy theory |
| **Emotional Trigger** | Entertainment, storytelling | Medium | Medium | Emotional contagion studies |
| **Social Proof/Authority** | New viewers, credibility-building | Medium | Medium | Social proof psychology |
| **Pattern Reversal** | Platform-native viewers, trend-aware | High | High | Surprise-attention link |
| **Direct Question** | Low-intent scrollers, engagement-seeking | Universal | Low | Question-response reflex |
| **Statistical Shock** | Data-driven niches, educational | Medium | Medium | Numerical attention research |

**Strategy Selection Rules:**
1. Match to audience's content intent state (from Stage 1)
2. Align with platform expertise level (native viewers expect platform-native hooks)
3. Consider niche sophistication (experts reject superficial hooks)
4. Respect attention allocation (low-intent needs stronger immediate hook)

### 2. Hook Generation Engine
Generate 6-8 distinct hook options using these templates:

#### Template Set A: Visual-First Hooks (TikTok/Reels optimized)
```
1. [Visual Action] + [Contradictory Audio] → "Wait, what just happened?"
2. [Extreme Close-up] + [Whispered Secret] → Intrigue through intimacy
3. [Sudden Motion] + [Frozen Frame] + [Question] → Pattern interrupt + curiosity
4. [Before/After Split] → Instant value demonstration
5. [Unexpected Object] + [Relevant Connection] → Novelty bridge to content
```

#### Template Set B: Audio-First Hooks (All platforms)
```
1. "[Number] things [target audience] gets wrong about [topic]" → Curiosity + relevance
2. "Stop scrolling if [pain point]" → Immediate relevance filter
3. "This is why [common belief] is wrong" → Contrarian position
4. "In the next [short timeframe], I'll show [specific outcome]" → Clear value promise
5. "[Shocking statistic] [target audience] people don't know" → Numerical surprise
```

#### Template Set C: Conceptual Hooks (YouTube Shorts optimized)
```
1. "[Expert View] vs [Popular View]: The truth about [topic]" → Authority + controversy
2. "What nobody tells you about [topic]" → Insider knowledge appeal
3. "The [adjective] mistake that's costing [target audience] [stakes]" → Fear of missing out
4. "If you want [desired outcome], stop [common action]" → Counterintuitive advice
5. "[Timeframe] to [transform] without [expected cost]" → Attractive impossibility
```

#### Template Set D: Engagement Hooks (Comment-driven)
```
1. "Comment [X] if [condition], [Y] if [opposite]" → Binary engagement prompt
2. "I bet [percentage] of [audience] doesn't know [fact]" → Challenge/statistics combo
3. "Which [option] is better? I'll test both" → Interactive demonstration
4. "This or that: [choice A] or [choice B]?" → Low-friction participation
5. "Save this for [future scenario]" → Share/save motivation
```

### 3. 3-Second Hook Rule Application
Score each generated hook against the 3-Second Hook Rule using this rubric:

| Dimension | Weight | Evaluation Criteria | Score (0-5) |
|-----------|--------|---------------------|-------------|
| **Instant Pattern Interrupt** | 30% | Breaks scroll pattern in first 0.5s | 0-5 |
| **Immediate Value Signal** | 25% | Viewer understands why to watch by 1.5s | 0-5 |
| **Audio-Visual Sync** | 20% | Sound and image reinforce each other | 0-5 |
| **Platform Native Feel** | 15% | Feels natural, not jarring to platform users | 0-5 |
| **Audience Match** | 10% | Appropriate for detected expertise level | 0-5 |

**Scoring Guide:**
- **5 (Exceptional):** Executes dimension perfectly, creates involuntary attention capture
- **4 (Strong):** Executes well, minor optimization possible
- **3 (Competent):** Functional but not standout, adequate for retention
- **2 (Weak):** Flawed execution, likely loses attention
- **1 (Poor):** Major issues, actively harmful to retention
- **0 (Failure):** Dimension absent or counterproductive

### 4. Curiosity Gap Analysis
Evaluate each hook for curiosity gap strength using this framework:

| Curiosity Type | Mechanism | Ideal For | Risk Level |
|---------------|-----------|-----------|------------|
| **Information Gap** | "What don't I know?" | Educational, utility content | Low |
| **Resolution Gap** | "How will this resolve?" | Storytelling, demonstrations | Low |
| **Causal Gap** | "Why did that happen?" | Explanations, behind-the-scenes | Medium |
| **Competence Gap** | "Could I do that?" | Tutorials, transformations | Medium |
| **Validation Gap** | "Am I right about this?" | Opinion, commentary | High |
| **Paradox Gap** | "How can both be true?" | Intellectual content | High |

**Gap Scoring Rubric (0-5):**
- **5:** Creates compelling, specific gap that demands resolution
- **4:** Strong gap with clear resolution motivation
- **3:** Adequate gap, some viewers may not feel compelled
- **2:** Weak gap, too vague or already known
- **1:** Minimal gap, no real curiosity created
- **0:** No gap or reveals too much, eliminates curiosity

### 5. Pattern Interrupt Effectiveness
Score each hook's pattern interrupt strength:

| Interrupt Type | Platform Fit | Novelty Half-Life | Production Complexity |
|----------------|--------------|-------------------|----------------------|
| **Visual Shock** | TikTok > Reels > Shorts | Very Short | Low |
| **Audio Surprise** | All platforms (equal) | Short | Low |
| **Motion Break** | TikTok > Reels > Shorts | Short | Medium |
| **Format Twist** | Shorts > TikTok > Reels | Medium | High |
| **Expectation Violation** | All platforms (equal) | Medium | High |
| **Context Subversion** | Shorts > TikTok > Reels | Long | Very High |

**Scoring Considerations:**
- **Immediate Impact (0-5):** How strongly it breaks the scroll pattern
- **Sustainability (0-5):** Whether the interrupt supports continued viewing
- **Appropriateness (0-5):** Whether the interrupt feels natural, not forced
- **Production Feasibility (0-5):** Practical implementation requirements

### 6. Framework Lock and Weight Selection
Based on hook analysis, determine the evaluation framework and dimension weights for retention scoring:

**Primary Framework Selection:**
- **Hook-Retention-Payoff (HRP):** Always active (core methodology)

### 7. Hook Ranking and Recommendation
Score all generated hooks across all criteria and produce final ranking:

**Composite Score Calculation:**
```
Hook Score = (3-Second Rule × 0.35) + (Curiosity Gap × 0.25) + 
             (Pattern Interrupt × 0.25) + (Audience Match × 0.15)
```

**Output Classification:**
- **Tier A (9.0-10.0):** Production-ready, exceptional expected performance
- **Tier B (7.5-8.9):** Strong candidate, minor optimization recommended
- **Tier C (6.0-7.4):** Functional, backup option or A/B test candidate
- **Tier D (below 6.0):** Not recommended, major revisions needed

## Failure Modes and Recovery

**Weak Hook Generation Recovery:**
If generated hooks score below 6.0 consistently:
1. Re-examine audience profile for missed opportunities
2. Test additional hook strategies from framework
3. Consider hybrid hook combinations
4. Recommend content restructuring if hooks are fundamentally limited

**Audience-Hook Mismatch Recovery:**
If no hooks match audience profile adequately:
1. Flag the mismatch clearly in recommendations
2. Suggest content adjustments for better hook options
3. Provide "best available" options with clear caveats
4. Recommend testing multiple approaches

## Framework Application Notes

**Hook-Retention-Payoff (HRP) Model:**
- **Hook Phase (0-3 seconds):** All scoring focuses on this critical window
- **Quality Threshold:** Hooks scoring below 3.0/5.0 jeopardize entire video performance
- **Transition Strength:** Hook must seamlessly flow into retention content

**3-Second Hook Rule Integration:**
- **Pattern Interrupt:** Must occur in first 0.5 seconds for visual-first platforms
- **Value Signal:** Must be clear by 1.5 seconds or viewers abandon
- **Decision Point:** Viewer commit-or-abandon decision happens by 3 seconds

**Curiosity Gap Theory:**
- **Gap Creation:** Hook must create specific, compelling gap
- **Gap Maintenance:** Content must sustain gap until resolution
- **Gap Resolution:** Payoff must deliver satisfying resolution

**Pattern Interrupt Theory:**
- **Interrupt Recognition:** Viewer must consciously notice pattern break
- **Interrupt Relevance:** Must be meaningfully connected to content
- **Interrupt Integration:** Must feel intentional, not random
